Voeg de Scootflow widget toe door middel van deze officiele Scootflow plugin.

== Description ==
Scootflow is het nieuwste en meest geavanceerde systeem voor scooterzaken. Het is ontwikkeld in het technologische hart van Nederland en wordt iedere dag beter.

== Installation ==
Wanneer je de plugin hebt geinstalleerd komt er een nieuw kopje met de titel "Scootflow" in je Wordpress admin paneel. Hier kun je je unieke bedrijfscode invullen.
Na het invullen van je unieke bedrijfscode word de widget automatisch zichtbaar op de website waar de plugin geinstalleerd is.
